Installation

There are two kinds of Frodo distributions. The source distributions (FrodoV?_?.Src.*) contain source code that has to be compiled on your system. The binary distributions (FrodoV?_?.<system name>.*) contain an executable that can be started directly.

Compiling under Unix

To build Frodo from an unpacked source distribution:

$ ./autogen.sh
$ make

For optimum performance, Frodo needs the following prerequisites:

  • SDL 2.x
  • GTK 3
autogen.sh and configure accept optional arguments (type ./autogen.sh --help for a full list).

The Frodo GUI requires the Frodo.ui file to be present in the directory /usr/local/share/Frodo (you can specify a different installation directory with ./configure --prefix). Either copy this file manually from src/Frodo.ui or become root and type

# make install

Compiling under BeOS

There are separate Makefiles for Frodo (Makefile.BeOS) and Frodo Lite (Makefile.BeOS.Lite). Type

$ cd src
$ make -fMakefile.BeOS
$ make -fMakefile.BeOS.Lite

to compile both versions.
